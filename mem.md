===========================
NVIM commands
===========================

CTRL-W v - дублировать окно справа - Like :split, but split vertically.

CTRL-W n - Открыть новое пустное окно внизу - Create a new window and start editing an empty file in it.

CTRL-W q - закрыть открытое окно
